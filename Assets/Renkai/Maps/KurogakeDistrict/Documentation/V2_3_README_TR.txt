RENKAI V2.3 UPDATE PATCH
========================

Bu ZIP bir güncelleme paketidir.

Kullanım:
1) ZIP'i çıkar.
2) İçindeki Assets klasörünü mevcut çalışan Renkai projenin üstüne kopyala.
   Önerilen proje: V2.2.1 Compile Fix veya PinkFix ile çalışan proje.
3) Unity açılınca üst menü:
   Renkai > V2.3 Install Character Reload Arena Update
4) Play'e bas.

Kontroller:
- Mouse1 = ateş
- R = reload
- V = quick sword slash
- C veya Left Ctrl = crouch
- F = plant testi

Düzeltmeler:
- Ctrl ateş etmez; sadece crouch olur.
- R artık respawn değil reload.
- Harita dışına düşmeyi azaltmak için safety floor ve sınır duvarları ekler.
- Yüklenen Meshy AI karakterleri Assets/Renkai/Characters içine eklendi.
- Character showcase kurulur.
