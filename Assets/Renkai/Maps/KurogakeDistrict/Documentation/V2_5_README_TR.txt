RENKAI V2.5 - BOT AI + PINK FIX + BETTER SPAWNS
================================================

Kurulum:
1) ZIP'i çıkar.
2) İçindeki Assets klasörünü mevcut çalışan Renkai projesinin ana klasörüne kopyala.
3) Üzerine yazma sorarsa EVET de.
4) Unity açılınca üst menüden:
   Renkai > V2.5 Install Bot AI + Pink Fix + Better Spawns
5) Play'e bas.

Bu sürümde:
- Pembe görünen sahne objelerine koyu neon/URP materyal atanır.
- 5v5 ekipler daha düzgün konumlarda spawn olur.
- Düşman botlar sana ateş eder.
- Ally botlar da karşı tarafa ateş eder.
- Takım renk halkaları ve isim etiketleri eklenir.
- Round + bomb sistemi korunur.
- Ölünce round sonuna kadar bekleme devam eder.

Kontroller:
1 = Rifle
2 = Pistol
3 = Sword
Mouse1 = ateş / sword seçiliyken slash
V = hızlı sword slash
R = reload
F = plant / defuse
C veya Ctrl = crouch

Notlar:
- Bu hâlâ local prototype. Online multiplayer ve gelişmiş pathfinding yok.
- Botlar basit line-of-sight combat AI kullanır.
- GLB model varsa ajanlara görsel olarak takılır; yoksa takım renkli hitbox ile yine çalışır.
