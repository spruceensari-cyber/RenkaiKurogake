ART PASS V3 İÇİN DEĞİŞTİRİLECEKLER:

- Placeholder roofs -> Blender/asset store shrine roof modules.
- Cover boxes -> sci-fi crates / stone shrine covers.
- Neon signs -> proper mesh + texture neon signage.
- Portal disc -> Shader Graph + particle system.
- Sakura tree placeholder -> optimized tree model.
- Wet floor -> PBR wet stone material + reflection probes.
- Lighting -> baked GI, light probes, reflection probes, occlusion culling.

PERFORMANCE TARGET:
- PC demo 60+ FPS.
- Static batching enabled.
- Mesh collider only for large simple surfaces; use box colliders for cover.
- LOD groups for buildings/props.
- Occlusion culling after final layout.
