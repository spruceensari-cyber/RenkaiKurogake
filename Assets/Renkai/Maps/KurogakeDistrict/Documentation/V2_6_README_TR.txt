RENKAI V2.6 - SERIOUS GAME MODE PATCH
=====================================

Kurulum:
1) ZIP'i çıkar.
2) İçindeki Assets klasörünü mevcut çalışan Renkai projesinin ana klasörüne kopyala.
3) Üzerine yazma sorarsa EVET de.
4) Unity'ye dön.
5) Üst menüden:
   Renkai > V2.6 Install Serious Game Mode
6) Play'e bas.

Eklenenler:
- Düşman botlar ateş eder.
- Ally botlar da ateş eder.
- Botlar basit strafe yapar.
- Botlar görüş hattı varsa ateş eder.
- World-space HP barlar.
- Bot isimleri.
- Takım renkleri.
- HP UI.
- Ammo UI.
- Weapon UI.
- Kill feed.
- Round score.
- Attackers/Defenders canlı sayısı.
- Bomb plant/defuse status UI.
- Ölünce round sonuna kadar bekleme.
- Round bitince herkes resetlenir.
- Pembe/bozuk görünen map objelerine daha güvenli materyal atanır.

Kontroller:
1 = Rifle
2 = Pistol
3 = Sword
Mouse1 = ateş / sword seçiliyken slash
V = hızlı sword slash
R = reload
F = plant / defuse
C veya Ctrl = crouch

Not:
Bu hâlâ local prototype. Gerçek online 5v5 değildir.
Ama artık "ciddi oyun modu" iskeleti oluştu:
round, skor, HP, bot savaşı, bomb, silah/kılıç seçimi.
