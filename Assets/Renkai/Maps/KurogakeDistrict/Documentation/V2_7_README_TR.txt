RENKAI V2.7 - AUTONOMOUS BOTS + AGENT VISUALS
=============================================

Kurulum:
1) ZIP'i çıkar.
2) İçindeki Assets klasörünü mevcut çalışan Renkai projesinin ana klasörüne kopyala.
3) Üzerine yazma sorarsa EVET de.
4) Unity'ye dön.
5) Üst menüden:
   Renkai > V2.7 Install Autonomous Bots + Agent Visuals
6) Play'e bas.

Bu sürüm ne ekler?
- Botlar artık daha özerk davranır.
- Duvar arkasından seni direkt bilmezler.
- Önce görüş açısına girmen gerekir.
- Seni görürse son konumunu hatırlar.
- Görüş kaybolursa investigate state'e geçer.
- Defenders bomb plant olursa bomb alanına rotate eder.
- Attackers A/B/Mid rotalarından ilerler.
- Botlar strafe yapar, görüş hattı varsa ateş eder.
- Agentler artık düz capsule değil; stylized hero-shooter gövdesi, helmet, visor, omuz, rifle, sırt kılıcı ile görünür.
- Roller: Duelist, Controller, Sentinel, Initiator, Blade.

Not:
Bu gerçek insan zekâsı veya online multiplayer değildir.
Ama önceki dummy AI'dan çok daha ileri, algı + hafıza + state machine kullanan tactical bot prototipidir.

Sonraki hedef V2.8:
- NavMesh tabanlı gerçek pathfinding.
- Cover seçimi.
- Flash/smoke skill.
- Q/E/C/X agent skill sistemi.
- Buy menu.
